# Android_OS
## Android とは
- 世界シェアNo.1のモバイルデバイス向けOS（iOS以上）
- オープンソースで公開されており、カスタムすることが可能
- 各Androidスマートフォンメーカーは、Androidを独自にカスタムしたOSをデバイスに組み込んでいる
  - サムスン：One UI
  - ASUS：ZenUI
  - シャオミ：MIUI
  - OPPO：ColorOS
  - ファーウェイ：HarmonyOS
  - Amazon：Fire OS
